# lotosoft

Football statistics analysis tool.

Collects and processes match data for statistical analysis.

## Data structure

```
data/
  grid-7/
    2025-2026/
      record-36.json
      ...
  grid-15/
    ...
state/
  grid-7.json
  grid-15.json
```

## Usage

```bash
pip install -r requirements.txt
python app.py --type grid-15
```
